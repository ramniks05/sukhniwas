# Color Scheme Options

The website currently uses a **Modern Teal/Green** color scheme. Here are alternative color schemes you can use:

## Current Scheme: Modern Teal/Green
```css
--primary-color: #0d9488;     /* Teal 600 */
--primary-dark: #0f766e;      /* Teal 700 */
--primary-light: #14b8a6;     /* Teal 500 */
--secondary-color: #f97316;   /* Orange 500 */
--accent-color: #ec4899;      /* Pink 500 */
```

## How to Change Colors

1. Open `assets/css/style.css`
2. Find the `:root` section at the top
3. Replace the color values with your chosen scheme
4. Save and refresh the website

## Tips for Choosing Colors

- **Primary Color**: Main brand color (buttons, links, highlights)
- **Secondary Color**: Accent color (call-to-actions, special elements)
- **Accent Color**: Third color (decorative elements, hover states)
- **Background Light**: Very light tint of primary color
- **Border Color**: Light tint of primary color

## Color Tools

- **Coolors.co**: Generate color palettes
- **Adobe Color**: Color wheel and harmony rules
- **Tailwind Colors**: Pre-built color palettes
